# xkb-ru-old

Cyrillic Slavic keyboard layout for xkb. It can be used as Old Church Slavonic, Russian pre-reform (before 1918), Bulgarian pre-reform (before 1945), Serbian, Macedonian, Ukrainian and Bielorussian layout. Based on 

For using place `xkb/symbols/ru@old` to `$HOME/.config/xkb/symbols/ru@old`
and run:
```sh
xkbcomp $HOME/.config/xkb/symbols/ru@old $DISPLAY
```

You may use various methods for autostarting scripts depending on your
desktop environment, see: https://wiki.archlinux.org/index.php/Autostarting.
